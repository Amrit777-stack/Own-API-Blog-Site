# Own API Blog Site

A full-stack blog application powered by a custom-built REST API.

## Features

* Create blog posts
* Read blog posts
* Update existing blog posts
* Delete blog posts
* Demonstrates REST API principles
* Built using Node.js and Express.js

## Tech Stack

* Node.js
* Express.js
* EJS
* REST APIs
* HTML
* CSS
* JavaScript

## Purpose

This project was built to practice full-stack development and RESTful API concepts by implementing CRUD operations in a blog application.
